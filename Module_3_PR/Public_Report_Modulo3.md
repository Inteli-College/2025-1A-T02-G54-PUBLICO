# Nautic.Bot - Relatório Público Resumido

## Resumo Executivo

A **Nautic.Bot** é uma startup brasileira de tecnologia náutica que desenvolve turbinas aquáticas portáteis recarregáveis com bateria de lítio, controladas remotamente via rádio. O projeto representa uma inovação significativa no setor de equipamentos aquáticos, oferecendo mobilidade subaquática acessível para esportes aquáticos, mergulho recreativo, turismo náutico e operações de salvamento marítimo.

Os principais destaques do projeto incluem uma turbina aquática controlada sem fio, preços competitivos em relação às soluções importadas, suporte técnico nacional e modelo de receita recorrente, design modular para fácil manutenção, e um mercado-alvo bilionário.

---

## 1. Produto

A turbina aquática subnado já é um produto existente e funcional. Sistema controlado remotamente com ESP32-C3 Super Mini, comunicação sem fio de longo alcance, motor brushless, operação em profundidades adequadas. Controle via joysticks + botões e  sistema de segurança.
